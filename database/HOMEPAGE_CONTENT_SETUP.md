# إعداد محتوى الصفحة الرئيسية

## نظرة عامة

تم إنشاء نظام لإدارة محتوى الصفحة الرئيسية من CMS. الآن يمكنك تعديل جميع أقسام الصفحة الرئيسية مباشرة من لوحة التحكم.

## الخطوات

### 1. إنشاء الجداول في قاعدة البيانات

قم بتشغيل ملف SQL التالي:

```powershell
psql -U postgres -d association_db -f admin-cms/database/create_homepage_content_table.sql
```

هذا سينشئ الجداول التالية:
- `homepage_content` - محتوى الأقسام المختلفة
- `homepage_stats` - الإحصائيات
- `homepage_values` - القيم والمبادئ
- `homepage_partners` - الشركاء

### 2. الوصول إلى صفحة الإدارة

1. سجّل الدخول إلى CMS: `http://localhost:3001`
2. من القائمة الجانبية، اختر **"الصفحة الرئيسية"**
3. ستجد تبويبات مختلفة لإدارة كل قسم

## الأقسام القابلة للتعديل

### 1. قسم البطل (Hero)
- العنوان الرئيسي
- العنوان الفرعي
- الوصف
- نص ورابط الزر الأول

### 2. عن الجمعية
- العنوان الفرعي
- العنوان الرئيسي
- الوصف الأول والثاني
- رابط الصورة
- نص ورابط الزر
- سنوات العطاء (في metadata)

### 3. الإحصائيات
- يمكنك إضافة/تعديل/حذف إحصائيات
- كل إحصائية تحتوي على:
  - التسمية (مثل: المستفيدين)
  - القيمة (مثل: 15,000+)
  - اسم الأيقونة (مثل: Users, Target, Heart, Award)

### 4. القيم
- يمكنك إضافة/تعديل/حذف القيم
- كل قيمة تحتوي على:
  - العنوان
  - الوصف
  - اسم الأيقونة

### 5. الشركاء
- يمكنك إضافة/تعديل/حذف الشركاء
- كل شريك يحتوي على:
  - الاسم
  - رابط الشعار
  - رابط الموقع (اختياري)

### 6. دعوة للعمل (CTA)
- العنوان
- الوصف
- نص ورابط الزر

## الأيقونات المتاحة

الأيقونات المدعومة حالياً:
- `Heart` - القلب
- `Shield` - الدرع
- `Users` - المستخدمين
- `Target` - الهدف
- `Award` - الجائزة
- `Building2` - المبنى
- `TrendingUp` - الاتجاه الصاعد

## ملاحظات مهمة

1. **المشاريع**: المشاريع المعروضة في الصفحة الرئيسية تأتي من جدول `projects` حيث `show_on_home = true`. يمكنك إدارتها من صفحة "المشاريع" في CMS.

2. **القيم الافتراضية**: إذا لم يتم تحميل المحتوى من API، سيتم استخدام القيم الافتراضية المضمنة في الكود.

3. **الحفظ**: بعد تعديل أي قسم، اضغط على زر **"حفظ جميع التغييرات"** في أسفل الصفحة.

4. **الترتيب**: يمكنك تحديد `order_index` لترتيب العناصر (الإحصائيات، القيم، الشركاء).

## API Endpoints

### Public API (للموقع العام)
```
GET /api/public/homepage
```

### Admin API (لإدارة المحتوى)
```
GET /api/homepage
PUT /api/homepage
```
