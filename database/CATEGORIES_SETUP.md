# إعداد نظام إدارة التصنيفات

## الخطوات المطلوبة:

### 1. إنشاء جدول التصنيفات في قاعدة البيانات

قم بتشغيل ملف SQL التالي:

```powershell
psql -U postgres -d association_db -f admin-cms\database\create_categories_table.sql
```

أو يدوياً في psql:

```sql
\c association_db
\i admin-cms/database/create_categories_table.sql
```

### 2. إعادة تشغيل CMS

```powershell
cd admin-cms
npm run dev
```

### 3. الوصول إلى صفحة التصنيفات

بعد تسجيل الدخول إلى CMS، ستجد رابط "تصنيفات الأخبار" في القائمة الجانبية.

## المميزات:

✅ **CRUD كامل** - إنشاء، قراءة، تحديث، وحذف التصنيفات
✅ **إدارة الألوان** - اختيار لون لكل تصنيف
✅ **ترتيب العرض** - تحديد ترتيب التصنيفات
✅ **تفعيل/إلغاء تفعيل** - إمكانية إخفاء تصنيفات معينة
✅ **منع الحذف** - لا يمكن حذف تصنيف مستخدم في أخبار موجودة

## التصنيفات الافتراضية:

سيتم إنشاء التصنيفات التالية تلقائياً:
- الفعاليات (events) - أخضر
- البرامج (programs) - أزرق
- الإنجازات (achievements) - برتقالي

## ملاحظات:

- يمكنك تعديل أو حذف التصنيفات الافتراضية
- عند إنشاء تصنيف جديد، سيتم إنشاء slug تلقائياً من الاسم
- يمكنك تغيير اللون والترتيب في أي وقت
