# إعداد صفحة التبرع

## نظرة عامة

تم إنشاء نظام إدارة صفحة **التبرع** التي تحتوي على:
- **معلومات عن التبرع** (Hero section ونص تمهيدي)
- **طرق التبرع** (حسابات بنكية، محافظ إلكترونية، إلخ)
- **منتجات التبرع** (مرتبطة بجدول `donation_products` الموجود)
- **أقسام متعددة** (الزكاة، الصدقة اليومية، دعم البرامج، الأوقاف، إلخ)

## الخطوات

### 1. إنشاء الجداول في قاعدة البيانات

قم بتشغيل ملف SQL التالي:

```powershell
psql -U postgres -d association_db -f admin-cms/database/create_donate_tables.sql
```

هذا سينشئ الجداول التالية:
- `donation_methods` - طرق التبرع (حسابات بنكية، محافظ إلكترونية)
- `donate_page_content` - محتوى الصفحة (Hero section وعناوين الأقسام)

**ملاحظة:** جدول `donation_products` موجود بالفعل ويتم استخدامه لعرض منتجات التبرع.

### 2. الوصول إلى صفحة الإدارة

1. سجّل الدخول إلى CMS: `http://localhost:3001`
2. من القائمة الجانبية، اختر **"التبرع"**
3. ستجد تبويبات:
   - **طرق التبرع** - لإدارة طرق التبرع (CRUD)
   - **محتوى الصفحة** - لإدارة Hero section وعناوين الأقسام

### 3. إدارة المحتوى

#### محتوى الصفحة
- **Hero Section:**
  - العنوان الرئيسي
  - الوصف
  - الحالة (نشط/غير نشط)
- **النص التمهيدي:**
  - النص التمهيدي
  - الحالة (نشط/غير نشط)
- **عناوين الأقسام:**
  - عنوان قسم منتجات التبرع
  - عنوان قسم الزكاة
  - عنوان قسم الصدقة اليومية
  - عنوان قسم دعم البرامج والمشاريع
  - عنوان قسم الأوقاف الخيرية
  - عنوان قسم الإهداءات الوقفية
- **قسم طرق التبرع:**
  - عنوان القسم
  - وصف القسم

#### إدارة طرق التبرع
- يمكنك إضافة/تعديل/حذف طرق التبرع
- كل طريقة تحتوي على:
  - الاسم
  - النوع (بنك، محفظة إلكترونية، أونلاين، أخرى)
  - رقم الحساب
  - اسم الحساب
  - اسم البنك
  - IBAN
  - SWIFT Code
  - QR Code (صورة)
  - اسم الأيقونة (اختياري)
  - الوصف
  - الترتيب
  - الحالة (نشط/غير نشط)

## استخدام الصفحة

بعد إتمام الإعداد، يمكن الوصول إلى صفحة التبرع من الموقع العام عبر:
- الرابط: `/membership-donate` (حسب التوجيه في `App.tsx`)

الصفحة تعرض تلقائياً:
- Hero slider مع العنوان والوصف
- النص التمهيدي
- منتجات التبرع من جدول `donation_products` (مجمعة حسب النوع)
- أقسام مختلفة (الزكاة، الصدقة اليومية، دعم البرامج، الأوقاف، الإهداءات)
- قسم طرق التبرع مع:
  - معلومات الحسابات البنكية
  - IBAN
  - QR Codes
  - أزرار نسخ للمعلومات

## ملاحظات مهمة

1. **منتجات التبرع:** يتم جلبها من جدول `donation_products` الموجود وتجميعها حسب `donation_type`:
   - `product` - منتجات التبرع
   - `zakat` - الزكاة
   - `sadaqa` - الصدقة اليومية
   - `support` - دعم البرامج والمشاريع
   - `waqf` - الأوقاف الخيرية
   - `giftWaqf` - الإهداءات الوقفية

2. **طرق التبرع:** يمكن إضافة طرق متعددة مع معلومات كاملة (حسابات بنكية، محافظ إلكترونية، إلخ).

3. **QR Codes:** يمكن رفع صور QR Code لكل طريقة تبرع لتسهيل عملية التبرع.

4. **الترتيب:** يتم ترتيب طرق التبرع باستخدام `order_index`.

5. **الأيقونات:** يمكن استخدام أيقونات من Lucide React (CreditCard, Wallet, Building, etc.).

6. **نسخ المعلومات:** في الواجهة الأمامية، يمكن للمستخدمين نسخ أرقام الحسابات وIBAN بسهولة.
