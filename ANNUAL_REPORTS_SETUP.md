# إعداد صفحة التقارير السنوية

## نظرة عامة

تم إنشاء نظام إدارة صفحة **التقارير السنوية** التي تحتوي على:
- **معلومات عن التقارير السنوية** (Hero section ونص تمهيدي)
- **قائمة التقارير السنوية** مع ملفات PDF
- **صورة غلاف** لكل تقرير (اختياري)

## الخطوات

### 1. إنشاء الجداول في قاعدة البيانات

قم بتشغيل ملف SQL التالي:

```powershell
psql -U postgres -d association_db -f admin-cms/database/create_annual_reports_tables.sql
```

هذا سينشئ الجداول التالية:
- `annual_reports` - التقارير السنوية
- `annual_reports_page_content` - محتوى الصفحة (Hero section ونص تمهيدي)

### 2. الوصول إلى صفحة الإدارة

1. سجّل الدخول إلى CMS: `http://localhost:3001`
2. من القائمة الجانبية، اختر **"التقارير السنوية"**
3. ستجد تبويبات:
   - **قائمة التقارير السنوية** - لإدارة التقارير (CRUD)
   - **محتوى الصفحة** - لإدارة Hero section والنص التمهيدي

### 3. إدارة المحتوى

#### محتوى الصفحة
- **Hero Section:**
  - العنوان الرئيسي
  - العنوان الفرعي
  - الحالة (نشط/غير نشط)
- **النص التمهيدي:**
  - النص التمهيدي
  - الحالة (نشط/غير نشط)

#### إدارة التقارير السنوية
- يمكنك إضافة/تعديل/حذف التقارير السنوية
- كل تقرير يحتوي على:
  - العنوان
  - الوصف (اختياري)
  - السنة
  - رابط ملف PDF
  - اسم الملف (يتم استخراجه تلقائياً)
  - صورة الغلاف (اختياري)
  - الترتيب
  - الحالة (نشط/غير نشط)

## الملفات المحدثة/المضافة

### قاعدة البيانات
- `admin-cms/database/create_annual_reports_tables.sql` - جداول التقارير السنوية

### API Routes
- `admin-cms/app/api/annual-reports/route.ts` - API لإدارة التقارير السنوية (GET, POST)
- `admin-cms/app/api/annual-reports/[id]/route.ts` - API لإدارة تقرير واحد (GET, PUT, DELETE)
- `admin-cms/app/api/annual-reports-page/route.ts` - API لإدارة محتوى الصفحة
- `admin-cms/app/api/public/annual-reports/route.ts` - Public API

### CMS Pages
- `admin-cms/app/annual-reports/page.tsx` - صفحة إدارة التقارير السنوية
- `admin-cms/app/annual-reports/new/page.tsx` - صفحة إضافة تقرير سنوي جديد
- `admin-cms/app/annual-reports/[id]/page.tsx` - صفحة تعديل تقرير سنوي

### Frontend
- `admin-cms/lib/apiClient.ts` - إضافة وظائف API لمحتوى الصفحة
- `src/lib/api.ts` - إضافة `getAnnualReports()`
- `src/pages/AnnualReports.tsx` - تحديث صفحة الواجهة الأمامية
- `admin-cms/components/AdminLayout.tsx` - إضافة رابط التقارير السنوية

## استخدام الصفحة

بعد إتمام الإعداد، يمكن الوصول إلى صفحة التقارير السنوية من الموقع العام عبر:
- الرابط: `/programs-annual` (حسب التوجيه في `App.tsx`)

الصفحة تعرض تلقائياً:
- Hero section مع العنوان والعنوان الفرعي
- النص التمهيدي
- قائمة التقارير السنوية مع:
  - صورة الغلاف (أو أيقونة افتراضية)
  - السنة
  - العنوان
  - الوصف
  - زر تحميل التقرير

## ملاحظات مهمة

1. **ملفات PDF:** يمكن رفع ملفات PDF أو إدخال رابط مباشر للملف.
2. **صورة الغلاف:** يمكن إضافة صورة غلاف لكل تقرير لعرض أفضل.
3. **الترتيب:** يتم ترتيب التقارير حسب السنة (تنازلياً) ثم حسب `order_index`.
4. **اسم الملف:** يتم استخراج اسم الملف تلقائياً من رابط الملف إذا لم يتم إدخاله يدوياً.
