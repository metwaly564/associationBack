# إعداد صفحة اللجان

## نظرة عامة

تم إنشاء نظام إدارة صفحة **اللجان** التي تحتوي على:
- **معلومات عن اللجان** (Hero section ونص تمهيدي)
- **قائمة اللجان** (دائمة ومؤقتة)
- **أعضاء كل لجنة**
- **مهام كل لجنة**

## الخطوات

### 1. إنشاء الجداول في قاعدة البيانات

قم بتشغيل ملف SQL التالي:

```powershell
psql -U postgres -d association_db -f admin-cms/database/create_committees_tables.sql
```

هذا سينشئ الجداول التالية:
- `committees` - اللجان
- `committee_members` - أعضاء اللجان
- `committee_tasks` - مهام اللجان
- `committees_page_content` - محتوى الصفحة (Hero section ونص تمهيدي)

### 2. الوصول إلى صفحة الإدارة

1. سجّل الدخول إلى CMS: `http://localhost:3001`
2. من القائمة الجانبية، اختر **"اللجان"**
3. ستجد تبويبات:
   - **قائمة اللجان** - لإدارة اللجان (CRUD)
   - **محتوى الصفحة** - لإدارة Hero section والنص التمهيدي

### 3. إدارة المحتوى

#### محتوى الصفحة
- **Hero Section:**
  - العنوان الرئيسي
  - العنوان الفرعي
  - الحالة (نشط/غير نشط)
- **النص التمهيدي:**
  - النص التمهيدي
  - الحالة (نشط/غير نشط)

#### إدارة اللجان
- يمكنك إضافة/تعديل/حذف اللجان
- كل لجنة تحتوي على:
  - اسم اللجنة
  - الوصف
  - النوع (دائمة/مؤقتة)
  - اسم الأيقونة (اختياري)
  - الترتيب
  - الحالة (نشط/غير نشط)
  - **أعضاء اللجنة:**
    - الاسم
    - الدور (مثال: رئيس اللجنة)
    - المنصب (مثال: عضو مجلس الإدارة)
    - الترتيب
  - **مهام اللجنة:**
    - نص المهمة
    - الترتيب

## الملفات المحدثة/المضافة

### قاعدة البيانات
- `admin-cms/database/create_committees_tables.sql` - جداول اللجان

### API Routes
- `admin-cms/app/api/committees/route.ts` - API لإدارة اللجان (GET, POST)
- `admin-cms/app/api/committees/[id]/route.ts` - API لإدارة لجنة واحدة (GET, PUT, DELETE)
- `admin-cms/app/api/committees-page/route.ts` - API لإدارة محتوى الصفحة
- `admin-cms/app/api/public/committees/route.ts` - Public API

### CMS Pages
- `admin-cms/app/committees/page.tsx` - صفحة إدارة اللجان
- `admin-cms/app/committees/new/page.tsx` - صفحة إضافة لجنة جديدة
- `admin-cms/app/committees/[id]/page.tsx` - صفحة تعديل لجنة

### Frontend
- `admin-cms/lib/apiClient.ts` - إضافة وظائف API لمحتوى الصفحة
- `src/lib/api.ts` - إضافة `getCommittees()`
- `src/pages/about/Committees.tsx` - تحديث صفحة الواجهة الأمامية
- `admin-cms/components/AdminLayout.tsx` - إضافة رابط اللجان

## استخدام الصفحة

بعد إتمام الإعداد، يمكن الوصول إلى صفحة اللجان من الموقع العام عبر:
- الرابط: `/about-committees` (حسب التوجيه في `App.tsx`)

الصفحة تعرض تلقائياً:
- Hero section مع العنوان والعنوان الفرعي
- النص التمهيدي
- قائمة اللجان مع:
  - اسم اللجنة
  - النوع (دائمة/مؤقتة)
  - الوصف
  - عدد الأعضاء (مع عرض أول 3 أعضاء)
  - عدد المهام (مع عرض أول مهمتين)
- قسم الجهات المشرفة (مع شريط تمرير تلقائي)

## ملاحظات مهمة

1. **الأعضاء:** يمكن إضافة أعضاء متعددين لكل لجنة مع تحديد الدور والمنصب.
2. **المهام:** يمكن إضافة مهام متعددة لكل لجنة.
3. **الأيقونات:** يمكن استخدام أيقونات من Lucide React (مثال: DollarSign, Stethoscope, Briefcase).
4. **الترتيب:** يتم ترتيب اللجان والأعضاء والمهام باستخدام `order_index`.
5. **النوع:** يمكن تحديد نوع اللجنة (دائمة أو مؤقتة) لعرضها بشكل مختلف في الواجهة الأمامية.
