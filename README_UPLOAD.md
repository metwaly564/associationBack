# نظام رفع الصور

## نظرة عامة

تم إضافة نظام شامل لرفع الصور في جميع أنحاء النظام. يمكنك الآن:
- رفع الصور من الجهاز
- إدخال روابط خارجية للصور
- معاينة الصور قبل الحفظ

## المكونات

### 1. ImageInput Component
مكون جديد (`components/ImageInput.tsx`) يوفر:
- زر لرفع الصور من الجهاز
- زر لإدخال رابط خارجي
- معاينة للصورة المرفوعة
- إمكانية حذف الصورة

### 2. Upload API
API endpoint جديد (`/api/upload`) يقوم بـ:
- استقبال الملفات المرفوعة
- التحقق من نوع وحجم الملف
- حفظ الملفات في `public/uploads/`
- إرجاع رابط الصورة

## الاستخدام

### في صفحة الصفحة الرئيسية

تم تحديث حقول الصور في صفحة "الصفحة الرئيسية" لاستخدام `ImageInput`:

```tsx
<Form.Item name={['section_about', 'image_url']} label="صورة القسم">
  <ImageInput placeholder="أدخل رابط الصورة أو ارفع صورة من الجهاز" />
</Form.Item>
```

### في صفحات أخرى

يمكنك استخدام `ImageInput` في أي مكان:

```tsx
import { ImageInput } from '@/components/ImageInput';

<Form.Item name="image_url" label="الصورة">
  <ImageInput />
</Form.Item>
```

## الملفات المحفوظة

- الملفات المرفوعة تُحفظ في: `public/uploads/`
- اسم الملف: `{timestamp}-{random}.{ext}`
- الرابط العام: `/uploads/{filename}`

## القيود

- الحد الأقصى لحجم الملف: 5MB
- الصيغ المدعومة: JPG, PNG, WebP, GIF
- للملفات الأخرى (PDF): استخدم `UploadField`

## تحديثات أخرى

- تم تحديث `UploadField` ليدعم الرفع الفعلي أيضاً
- جميع الصفحات التي تستخدم حقول الصور يمكن تحديثها لاستخدام `ImageInput`
