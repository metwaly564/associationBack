# إعداد صفحة الإطار الاستراتيجي

## نظرة عامة

تم إنشاء نظام شامل لإدارة صفحة **الإطار الاستراتيجي** التي تحتوي على:
- **الرؤية** (Vision)
- **الرسالة** (Mission)
- **القيم** (Values)
- **الأهداف الاستراتيجية** (Strategic Goals)

## الخطوات

### 1. إنشاء الجداول في قاعدة البيانات

قم بتشغيل ملفات SQL التالية بالترتيب:

```powershell
# 1. إنشاء جدول القيم (إذا لم يكن موجوداً)
psql -U postgres -d association_db -f admin-cms/database/add_values_to_about_sections.sql

# 2. إنشاء جدول محتوى صفحة الإطار الاستراتيجي
psql -U postgres -d association_db -f admin-cms/database/create_strategic_framework_page_table.sql
```

هذا سينشئ/يحدث الجداول التالية:
- `about_values` - القيم (إذا لم يكن موجوداً)
- `strategic_framework_page_content` - محتوى صفحة الإطار الاستراتيجي (Hero section)

**ملاحظة:** الجداول `about_sections` و `about_goals` موجودة مسبقاً من إعداد صفحة "من نحن".

### 2. الوصول إلى صفحة الإدارة

1. سجّل الدخول إلى CMS: `http://localhost:3001`
2. من القائمة الجانبية، اختر **"الإطار الاستراتيجي"**
3. ستجد تبويب لإدارة محتوى Hero section

### 3. إدارة المحتوى من صفحة "الصفحات الثابتة"

يمكنك أيضاً إدارة الرؤية والرسالة والأهداف والقيم من:
- **الصفحات الثابتة** → تبويب **"أقسام صفحة 'من نحن'"**
- ستجد تبويبات:
  - **رؤيتنا** - العنوان والمحتوى والأيقونة
  - **رسالتنا** - العنوان والمحتوى والأيقونة
  - **أهدافنا الاستراتيجية** - قائمة الأهداف (يمكن إضافة/تعديل/حذف)
  - **قيمنا** - قائمة القيم (يمكن إضافة/تعديل/حذف)

## الأقسام القابلة للتعديل

### 1. Hero Section (من صفحة الإطار الاستراتيجي)
- العنوان الرئيسي
- العنوان الفرعي
- الحالة (نشط/غير نشط)

### 2. الرؤية (من الصفحات الثابتة)
- العنوان
- المحتوى
- اسم الأيقونة

### 3. الرسالة (من الصفحات الثابتة)
- العنوان
- المحتوى
- اسم الأيقونة

### 4. القيم (من الصفحات الثابتة)
- العنوان
- الوصف
- اسم الأيقونة
- الترتيب
- الحالة (نشط/غير نشط)

### 5. الأهداف الاستراتيجية (من الصفحات الثابتة)
- النص
- الترتيب
- الحالة (نشط/غير نشط)

## الأيقونات المتاحة

الأيقونات المدعومة من Lucide React:
- `Eye` - العين (لرؤيتنا)
- `Target` - الهدف (لرسالتنا)
- `Award` - الجائزة (للأهداف)
- `Heart` - القلب (للقيم)
- `Shield` - الدرع (للقيم)
- `Users` - المستخدمين
- `CheckCircle` - علامة صح
- `FileText` - الملف

## الملفات المحدثة/المضافة

### قاعدة البيانات
- `admin-cms/database/add_values_to_about_sections.sql` - إضافة جدول القيم
- `admin-cms/database/create_strategic_framework_page_table.sql` - جدول محتوى الصفحة

### API Routes
- `admin-cms/app/api/about/route.ts` - تحديث لدعم القيم
- `admin-cms/app/api/public/about/route.ts` - تحديث لدعم القيم
- `admin-cms/app/api/strategic-framework-page/route.ts` - API لإدارة محتوى الصفحة
- `admin-cms/app/api/public/strategic-framework/route.ts` - Public API

### CMS Pages
- `admin-cms/app/pages/page.tsx` - إضافة تبويب القيم
- `admin-cms/app/strategic-framework/page.tsx` - صفحة إدارة محتوى الإطار الاستراتيجي

### Frontend
- `admin-cms/lib/apiClient.ts` - إضافة وظائف API
- `src/lib/api.ts` - إضافة `getStrategicFramework()`
- `src/pages/about/StrategicFramework.tsx` - صفحة الواجهة الأمامية الكاملة

### Navigation
- `admin-cms/components/AdminLayout.tsx` - إضافة رابط في القائمة الجانبية

## استخدام الصفحة

بعد إتمام الإعداد، يمكن الوصول إلى صفحة الإطار الاستراتيجي من الموقع العام عبر:
- الرابط: `/about-strategy` (حسب التوجيه في `App.tsx`)

الصفحة تعرض تلقائياً:
- Hero section مع العنوان والعنوان الفرعي
- الرؤية والرسالة جنباً إلى جنب
- القيم في شبكة من البطاقات
- الأهداف الاستراتيجية كقائمة مرقمة

## ملاحظات مهمة

1. **القيم والأهداف:** يمكن إدارتها من صفحة "الصفحات الثابتة" → "أقسام صفحة 'من نحن'"
2. **Hero Section:** يمكن إدارته من صفحة "الإطار الاستراتيجي" في CMS
3. **الرؤية والرسالة:** مشتركة بين صفحة "من نحن" وصفحة "الإطار الاستراتيجي"
4. **الأيقونات:** تأكد من استخدام أسماء الأيقونات الصحيحة من Lucide React
