# إعداد إظهار الأخبار والمشاريع المميزة في الصفحة الرئيسية

## نظرة عامة

تم إضافة إمكانية تمييز الأخبار والمشاريع لإظهارها في الصفحة الرئيسية مع إمكانية تحديد الأولوية.

## الخطوات

### 1. تحديث قاعدة البيانات للأخبار

قم بتشغيل ملف SQL التالي لإضافة الحقول المطلوبة:

```powershell
psql -U postgres -d association_db -f admin-cms/database/add_homepage_fields_to_news.sql
```

هذا سيضيف الحقول التالية لجدول `news`:
- `show_on_home` - إظهار في الصفحة الرئيسية (BOOLEAN)
- `priority` - الأولوية (INTEGER)

**ملاحظة**: جدول `projects` يحتوي بالفعل على هذه الحقول.

### 2. الوصول إلى الإعدادات

#### للأخبار:
1. افتح CMS: `http://localhost:3001`
2. اذهب إلى **"الأخبار"**
3. اضغط على **"تعديل"** لأي خبر
4. في tab **"النشر"** ستجد:
   - **إظهار في الصفحة الرئيسية**: مفتاح تشغيل/إيقاف
   - **أولوية العرض**: رقم (كلما زاد الرقم، كلما ظهر أولاً)

#### للمشاريع:
1. افتح CMS: `http://localhost:3001`
2. اذهب إلى **"المشاريع والبرامج"**
3. اضغط على **"تعديل"** أو **"إنشاء مشروع"**
4. في أسفل النموذج ستجد:
   - **إظهار في الرئيسية**: مفتاح تشغيل/إيقاف
   - **أولوية العرض**: رقم

## كيفية الاستخدام

### للأخبار:
1. عند إنشاء أو تعديل خبر، فعّل **"إظهار في الصفحة الرئيسية"**
2. حدد **"أولوية العرض"** (مثلاً: 10، 20، 30)
3. احفظ التغييرات
4. ستظهر الأخبار المميزة في الصفحة الرئيسية مرتبة حسب الأولوية

### للمشاريع:
1. عند إنشاء أو تعديل مشروع، فعّل **"إظهار في الرئيسية"**
2. حدد **"أولوية العرض"** (مثلاً: 10، 20، 30)
3. احفظ التغييرات
4. ستظهر المشاريع المميزة في الصفحة الرئيسية مرتبة حسب الأولوية

## الملفات المحدثة

### قاعدة البيانات:
- `admin-cms/database/add_homepage_fields_to_news.sql` - SQL script لإضافة الحقول

### API Routes:
- `admin-cms/app/api/news/route.ts` - تحديث POST لدعم الحقول الجديدة
- `admin-cms/app/api/news/[id]/route.ts` - تحديث PUT لدعم الحقول الجديدة
- `admin-cms/app/api/public/news/route.ts` - تحديث GET لإرجاع الحقول الجديدة
- `admin-cms/app/api/public/homepage-featured/route.ts` - **جديد**: API للحصول على المحتوى المميز

### CMS Pages:
- `admin-cms/app/news/[id]/page.tsx` - إضافة حقول "إظهار في الرئيسية" و "الأولوية"
- `admin-cms/app/news/new/page.tsx` - إضافة حقول "إظهار في الرئيسية" و "الأولوية"
- `admin-cms/app/projects/[id]/page.tsx` - تحديث لحفظ البيانات بشكل صحيح
- `admin-cms/app/projects/new/page.tsx` - تحديث لحفظ البيانات بشكل صحيح

### Frontend:
- `src/lib/api.ts` - إضافة `getHomepageFeatured()`
- `src/pages/Home.tsx` - إضافة قسم الأخبار المميزة وتحديث قسم المشاريع

## الميزات

1. **إظهار في الصفحة الرئيسية**: يمكنك تحديد أي خبر أو مشروع يظهر في الصفحة الرئيسية
2. **الأولوية**: يمكنك تحديد ترتيب العرض (الأعلى أولوية يظهر أولاً)
3. **عرض تلقائي**: الصفحة الرئيسية تعرض تلقائياً:
   - آخر 3 أخبار مميزة (مرتبة حسب الأولوية)
   - آخر 3 مشاريع مميزة (مرتبة حسب الأولوية)

## بعد التطبيق

1. شغّل SQL script لإضافة الحقول للأخبار
2. افتح CMS وعدّل أي خبر أو مشروع
3. فعّل "إظهار في الصفحة الرئيسية" وحدد الأولوية
4. احفظ التغييرات
5. افتح الموقع العام: `http://localhost:5173`
6. ستجد الأخبار والمشاريع المميزة في الصفحة الرئيسية!

## ملاحظات مهمة

1. **الأولوية**: كلما زاد الرقم، كلما ظهر أولاً (مثلاً: 30 يظهر قبل 20)
2. **الحد الأقصى**: يتم عرض آخر 3 أخبار و 3 مشاريع فقط في الصفحة الرئيسية
3. **الترتيب**: يتم الترتيب أولاً حسب الأولوية (DESC)، ثم حسب تاريخ النشر/الإنشاء (DESC)
