# إعداد صفحة فريق العمل

## الخطوات المطلوبة

### 1. إنشاء جداول قاعدة البيانات

قم بتشغيل ملف SQL التالي لإنشاء الجداول المطلوبة:

```bash
psql -U postgres -d association_db -f admin-cms/database/create_team_page_table.sql
```

هذا الملف سينشئ:
- `team_page_content`: محتوى أقسام الصفحة (العناوين)
- `team_page_hero_images`: صور Hero slider
- `team_page_executive_director`: معلومات المدير التنفيذي

### 2. الوصول إلى إدارة صفحة فريق العمل

1. افتح لوحة التحكم: `http://localhost:3001`
2. انتقل إلى **فريق العمل** من القائمة الجانبية
3. ستجد تابين:
   - **قائمة الأعضاء**: لإدارة أعضاء الفريق (مجلس الإدارة وفريق العمل)
   - **محتوى الصفحة**: لإدارة محتوى الصفحة العامة

### 3. إدارة محتوى الصفحة

في تاب **محتوى الصفحة**، ستجد ثلاثة أقسام:

#### أ. صور Hero
- إضافة/حذف/ترتيب صور Hero slider
- كل صورة لها ترتيب عرض وحالة (نشط/غير نشط)

#### ب. المدير التنفيذي
- الاسم الكامل
- المسمى الوظيفي
- البريد الإلكتروني
- رقم الهاتف
- الصورة الشخصية
- الفترة (من - إلى)
- المؤهل العلمي
- الوصف

#### ج. العناوين
- عنوان قسم مجلس الإدارة
- عنوان قسم موظفو الجمعية

### 4. إدارة أعضاء الفريق

في تاب **قائمة الأعضاء**:
- يمكنك إضافة أعضاء جدد
- تعديل بيانات الأعضاء
- حذف الأعضاء
- تصفية حسب النوع (مجلس الإدارة / فريق العمل)
- البحث بالاسم أو المسمى

### 5. التحقق من الصفحة العامة

افتح الصفحة العامة: `http://localhost:5173/about/team`

ستجد:
- صور Hero slider (تتغير تلقائياً كل 4 ثوان)
- معلومات المدير التنفيذي (من CMS)
- قائمة أعضاء مجلس الإدارة (من قاعدة البيانات)
- قائمة موظفو الجمعية (من قاعدة البيانات)

## ملاحظات

- جميع البيانات تُقرأ ديناميكياً من قاعدة البيانات
- الصور يمكن رفعها عبر مكون `ImageInput`
- ترتيب الأعضاء يتم حسب `order_index`
- يمكن إخفاء أي صورة Hero أو عضو من خلال تعطيل الحالة
