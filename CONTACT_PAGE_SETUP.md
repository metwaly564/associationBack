# إعداد صفحة التواصل

## نظرة عامة

تم إنشاء نظام لإدارة صفحة التواصل من CMS. الآن يمكنك تعديل جميع محتويات الصفحة مباشرة من لوحة التحكم.

## الخطوات المطلوبة

### 1. إنشاء الجداول في قاعدة البيانات

قم بتشغيل ملف SQL التالي:

```bash
psql -U postgres -d association_db -f admin-cms/database/create_contact_page_table.sql
```

هذا سينشئ الجداول التالية:
- `contact_page_content` - محتوى أقسام الصفحة
- `contact_info` - معلومات الاتصال (هاتف، بريد، عنوان، وسائل التواصل)
- `contact_working_hours` - ساعات العمل

### 2. الوصول إلى صفحة الإدارة

1. افتح لوحة التحكم: `http://localhost:3001`
2. من القائمة الجانبية، اختر **"صفحة التواصل"**
3. ستجد تبويبات مختلفة لإدارة كل قسم

## الأقسام القابلة للتعديل

### 1. قسم Hero
- العنوان الرئيسي
- العنوان الفرعي

### 2. معلومات الاتصال
يمكنك إضافة/تعديل/حذف معلومات الاتصال:
- **الهاتف**: رقم الهاتف مع أيقونة (Phone)
- **البريد الإلكتروني**: عناوين البريد مع أيقونة (Mail)
- **العنوان**: معلومات العنوان مع أيقونة (MapPin)
- **وسائل التواصل**: روابط وسائل التواصل مع أيقونات (Facebook, Twitter, Instagram, LinkedIn)

كل عنصر يحتوي على:
- نوع المعلومات (phone, email, address, social)
- التسمية (اختياري)
- القيمة (مطلوب)
- اسم الأيقونة (اختياري)
- ترتيب العرض
- الحالة (نشط/غير نشط)

### 3. ساعات العمل
يمكنك إضافة/تعديل/حذف ساعات العمل:
- اليوم/الأيام (مثال: السبت - الأربعاء)
- الوقت (مثال: 8:00 ص - 4:00 م)
- حالة الإغلاق (مغلق/مفتوح)
- ترتيب العرض

## الأيقونات المتاحة

الأيقونات المدعومة من Lucide React:
- `Phone` - الهاتف
- `Mail` - البريد الإلكتروني
- `MapPin` - الموقع
- `Facebook` - فيسبوك
- `Twitter` - تويتر
- `Instagram` - إنستغرام
- `Linkedin` - لينكد إن
- `Clock` - الساعة (للساعات)

## ملاحظات مهمة

1. **التجميع التلقائي**: في الواجهة الأمامية، يتم تجميع معلومات الاتصال حسب النوع (phone, email, address, social) تلقائياً.

2. **الأيقونات**: إذا لم تحدد أيقونة، سيتم استخدام أيقونة افتراضية حسب نوع المعلومات.

3. **الحفظ**: بعد تعديل أي قسم، اضغط على زر **"حفظ جميع التغييرات"** في أسفل الصفحة.

4. **الترتيب**: يمكنك تحديد `order_index` لترتيب العناصر.

## API Endpoints

### Public API (للموقع العام)
```
GET /api/public/contact-page
```

### Admin API (لإدارة المحتوى)
```
GET /api/contact-page
PUT /api/contact-page
```

## الملفات المحدثة

- `admin-cms/database/create_contact_page_table.sql` - SQL script لإنشاء الجداول
- `admin-cms/app/api/contact-page/route.ts` - API لإدارة المحتوى
- `admin-cms/app/api/public/contact-page/route.ts` - Public API
- `admin-cms/app/contact/page.tsx` - صفحة إدارة المحتوى في CMS
- `admin-cms/lib/apiClient.ts` - إضافة وظائف API
- `src/lib/api.ts` - إضافة وظائف API للموقع العام
- `src/pages/Contact.tsx` - تحديث الصفحة لاستخدام المحتوى من API
- `admin-cms/components/AdminLayout.tsx` - إضافة رابط صفحة التواصل
