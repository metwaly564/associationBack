# إعداد محتوى صفحة "البرامج والمشاريع"

## نظرة عامة

تم إنشاء نظام لإدارة محتوى صفحة "البرامج والمشاريع" بشكل ديناميكي من CMS:
- **العنوان والوصف** في أعلى الصفحة
- **التصنيفات** (الفئات) للفلترة

## الخطوات

### 1. إنشاء الجداول في قاعدة البيانات

قم بتشغيل ملف SQL التالي:

```powershell
psql -U postgres -d association_db -f admin-cms/database/create_programs_page_table.sql
```

هذا سينشئ الجداول التالية:
- `programs_page_content` - محتوى الصفحة (العنوان، الوصف)
- `programs_page_categories` - تصنيفات البرامج

### 2. الوصول إلى صفحة الإدارة

1. سجّل الدخول إلى CMS: `http://localhost:3001`
2. من القائمة الجانبية، اختر **"المشاريع والبرامج"**
3. ستجد tabين:
   - **قائمة المشاريع**: لإدارة المشاريع الفردية
   - **محتوى الصفحة**: لإدارة محتوى الصفحة نفسها

## الأقسام القابلة للتعديل

### 1. محتوى الصفحة
- العنوان الرئيسي
- الوصف (العنوان الفرعي)

### 2. التصنيفات
- يمكنك إضافة/تعديل/حذف التصنيفات
- كل تصنيف يحتوي على:
  - مفتاح التصنيف (category_key): مثل `all`, `social`, `health`
  - اسم التصنيف (label): مثل "جميع البرامج"، "الرعاية الاجتماعية"
  - الترتيب (order_index)

## الملفات المحدثة

- `admin-cms/database/create_programs_page_table.sql` - SQL script
- `admin-cms/app/api/programs-page/route.ts` - API لإدارة المحتوى
- `admin-cms/app/api/public/programs-page/route.ts` - Public API
- `admin-cms/app/projects/page.tsx` - صفحة الإدارة في CMS (تمت إضافة tab)
- `admin-cms/lib/apiClient.ts` - إضافة وظائف API
- `src/lib/api.ts` - إضافة وظائف API للموقع العام
- `src/pages/Programs.tsx` - تحديث الصفحة لقراءة البيانات من API

## ملاحظات مهمة

1. **الحفظ**: بعد تعديل أي قسم، اضغط على زر **"حفظ جميع التغييرات"** في أسفل الصفحة
2. **الترتيب**: يمكنك تحديد `order_index` لترتيب التصنيفات
3. **مفتاح التصنيف**: يجب أن يكون فريداً (مثل: `all`, `social`, `health`, `education`, `employment`)

## بعد التطبيق

1. شغّل SQL script لإنشاء الجداول
2. افتح CMS واذهب إلى "المشاريع والبرامج" → tab "محتوى الصفحة"
3. عدّل المحتوى والتصنيفات
4. احفظ التغييرات
5. افتح الموقع العام: `http://localhost:5173/programs`
6. ستجد المحتوى محدثاً!
