# إعداد صفحة الهيكل التنظيمي

## نظرة عامة

تم تحديث نظام إدارة صفحة **الهيكل التنظيمي** التي تحتوي على:
- **معلومات عن الهيكل التنظيمي** (Hero section ونص تمهيدي)
- **صورة/مخطط الهيكل التنظيمي** (صورة توضيحية)
- **عناصر الهيكل** (قائمة المستويات الإدارية)

## الخطوات

### 1. إنشاء الجداول في قاعدة البيانات

قم بتشغيل ملف SQL التالي:

```powershell
psql -U postgres -d association_db -f admin-cms/database/create_org_structure_content_table.sql
```

هذا سينشئ الجدول التالي:
- `org_structure_content` - محتوى الصفحة (Hero section ونص تمهيدي وصورة)

**ملاحظة:** جدول `org_structure` موجود مسبقاً ويستخدم لعرض عناصر الهيكل التنظيمي.

### 2. الوصول إلى صفحة الإدارة

1. سجّل الدخول إلى CMS: `http://localhost:3001`
2. من القائمة الجانبية، اختر **"الهيكل التنظيمي"**
3. ستجد تبويبات:
   - **عناصر الهيكل** - لإدارة المستويات الإدارية (موجود مسبقاً)
   - **محتوى الصفحة** - لإدارة Hero section والنص التمهيدي والصورة

### 3. إدارة المحتوى

#### محتوى الصفحة
- **Hero Section:**
  - العنوان الرئيسي
  - العنوان الفرعي
  - الحالة (نشط/غير نشط)
- **النص التمهيدي:**
  - النص التمهيدي
  - الحالة (نشط/غير نشط)
- **صورة الهيكل التنظيمي:**
  - صورة/مخطط الهيكل التنظيمي
  - الحالة (نشط/غير نشط)

#### عناصر الهيكل
- يمكنك إضافة/تعديل/حذف عناصر الهيكل التنظيمي
- كل عنصر يحتوي على:
  - العنوان
  - الوصف
  - الترتيب (يتم تحديده تلقائياً)

## الملفات المحدثة/المضافة

### قاعدة البيانات
- `admin-cms/database/create_org_structure_content_table.sql` - جدول محتوى الصفحة

### API Routes
- `admin-cms/app/api/org-structure-page/route.ts` - API لإدارة محتوى الصفحة
- `admin-cms/app/api/public/org-structure/route.ts` - Public API (محدث)

### CMS Pages
- `admin-cms/app/org-structure/page.tsx` - تحديث صفحة إدارة الهيكل التنظيمي (إضافة تبويب محتوى الصفحة)

### Frontend
- `admin-cms/lib/apiClient.ts` - إضافة وظائف API لمحتوى الصفحة
- `src/lib/api.ts` - إضافة `getOrgStructure()`
- `src/pages/about/OrganizationalStructure.tsx` - تحديث صفحة الواجهة الأمامية

## استخدام الصفحة

بعد إتمام الإعداد، يمكن الوصول إلى صفحة الهيكل التنظيمي من الموقع العام عبر:
- الرابط: `/about-structure` (حسب التوجيه في `App.tsx`)

الصفحة تعرض تلقائياً:
- Hero section مع العنوان والعنوان الفرعي
- النص التمهيدي
- صورة/مخطط الهيكل التنظيمي (إن وجدت)
- عناصر الهيكل التنظيمي كقائمة مع وصف لكل مستوى
- زر للانتقال إلى صفحة فريق العمل

## ملاحظات مهمة

1. **العناصر:** يتم إدارتها من تبويب "عناصر الهيكل" في صفحة CMS.
2. **الصورة:** يمكن رفع صورة أو مخطط للهيكل التنظيمي من تبويب "محتوى الصفحة".
3. **الترتيب:** يتم ترتيب العناصر باستخدام `order_index` في جدول `org_structure`.
4. **التكامل:** الصفحة تحتوي على زر للانتقال إلى صفحة فريق العمل.
