# إعداد صفحة الوظائف

## نظرة عامة

تم إنشاء نظام إدارة صفحة **الوظائف** التي تحتوي على:
- **معلومات عن الوظائف** (Hero section ونص تمهيدي)
- **قائمة الوظائف المتاحة** مع تفاصيل كاملة
- **نموذج التقديم** للوظائف
- **إدارة طلبات التوظيف**

## الخطوات

### 1. إنشاء الجداول في قاعدة البيانات

قم بتشغيل ملف SQL التالي:

```powershell
psql -U postgres -d association_db -f admin-cms/database/create_jobs_tables.sql
```

هذا سينشئ الجداول التالية:
- `jobs` - الوظائف المتاحة
- `job_applications` - طلبات التوظيف
- `jobs_page_content` - محتوى الصفحة (Hero section ونص تمهيدي)

### 2. الوصول إلى صفحة الإدارة

1. سجّل الدخول إلى CMS: `http://localhost:3001`
2. من القائمة الجانبية، اختر **"الوظائف"**
3. ستجد تبويبات:
   - **قائمة الوظائف** - لإدارة الوظائف (CRUD)
   - **طلبات التوظيف** - لعرض طلبات التوظيف المستلمة
   - **محتوى الصفحة** - لإدارة Hero section والنص التمهيدي

### 3. إدارة المحتوى

#### محتوى الصفحة
- **Hero Section:**
  - العنوان الرئيسي
  - العنوان الفرعي
  - الحالة (نشط/غير نشط)
- **النص التمهيدي:**
  - النص التمهيدي
  - الحالة (نشط/غير نشط)
- **نموذج التقديم:**
  - عنوان النموذج
  - نص النموذج

#### إدارة الوظائف
- يمكنك إضافة/تعديل/حذف الوظائف
- كل وظيفة تحتوي على:
  - المسمى الوظيفي
  - القسم
  - الموقع
  - نوع التوظيف (دوام كامل/جزئي/عقد/تدريب)
  - الوصف الوظيفي
  - المتطلبات
  - المسؤوليات
  - المزايا
  - نطاق الراتب
  - آخر موعد للتقديم
  - الترتيب
  - الحالة (نشط/غير نشط)

## الملفات المحدثة/المضافة

### قاعدة البيانات
- `admin-cms/database/create_jobs_tables.sql` - جداول الوظائف

### API Routes
- `admin-cms/app/api/jobs/route.ts` - API لإدارة الوظائف (GET, POST)
- `admin-cms/app/api/jobs/[id]/route.ts` - API لإدارة وظيفة واحدة (GET, PUT, DELETE)
- `admin-cms/app/api/job-applications/route.ts` - API لإدارة طلبات التوظيف
- `admin-cms/app/api/jobs-page/route.ts` - API لإدارة محتوى الصفحة
- `admin-cms/app/api/public/jobs/route.ts` - Public API

### CMS Pages
- `admin-cms/app/jobs/page.tsx` - صفحة إدارة الوظائف
- `admin-cms/app/jobs/new/page.tsx` - صفحة إضافة وظيفة جديدة
- `admin-cms/app/jobs/[id]/page.tsx` - صفحة تعديل وظيفة

### Frontend
- `admin-cms/lib/apiClient.ts` - إضافة وظائف API لمحتوى الصفحة
- `src/lib/api.ts` - إضافة `getJobs()` و `submitJobApplication()`
- `src/pages/membership/Jobs.tsx` - تحديث صفحة الواجهة الأمامية
- `admin-cms/components/AdminLayout.tsx` - إضافة رابط الوظائف

## استخدام الصفحة

بعد إتمام الإعداد، يمكن الوصول إلى صفحة الوظائف من الموقع العام عبر:
- الرابط: `/membership-jobs` (حسب التوجيه في `App.tsx`)

الصفحة تعرض تلقائياً:
- Hero section مع العنوان والعنوان الفرعي
- النص التمهيدي
- قائمة الوظائف المتاحة مع:
  - المسمى الوظيفي
  - القسم والموقع
  - نوع التوظيف
  - نطاق الراتب
  - آخر موعد للتقديم
  - الوصف المختصر
- عند اختيار وظيفة، يتم عرض التفاصيل الكاملة
- نموذج التقديم مع:
  - الاسم الكامل
  - البريد الإلكتروني
  - رقم الهاتف
  - رابط السيرة الذاتية
  - رسالة التقديم

## ملاحظات مهمة

1. **طلبات التوظيف:** يتم حفظ جميع طلبات التوظيف في جدول `job_applications` ويمكن عرضها من تبويب "طلبات التوظيف" في CMS.
2. **السيرة الذاتية:** يمكن للمستخدمين إدخال رابط السيرة الذاتية (مثل Google Drive أو Dropbox).
3. **الترتيب:** يتم ترتيب الوظائف باستخدام `order_index`.
4. **نوع التوظيف:** يمكن تحديد نوع التوظيف (دوام كامل، جزئي، عقد، تدريب).
5. **آخر موعد:** يمكن تحديد آخر موعد للتقديم على الوظيفة.
