# إعداد أقسام صفحة "من نحن"

## نظرة عامة

تم إنشاء نظام لإدارة أقسام صفحة "من نحن" بشكل منفصل:
- **رؤيتنا**
- **رسالتنا**
- **أهدافنا الاستراتيجية**

## الخطوات

### 1. إنشاء الجداول في قاعدة البيانات

قم بتشغيل ملف SQL التالي:

```powershell
psql -U postgres -d association_db -f admin-cms/database/create_about_sections_table.sql
```

هذا سينشئ الجداول التالية:
- `about_sections` - رؤيتنا ورسالتنا
- `about_goals` - الأهداف الاستراتيجية

### 2. الوصول إلى صفحة الإدارة

1. سجّل الدخول إلى CMS: `http://localhost:3001`
2. من القائمة الجانبية، اختر **"صفحة "من نحن""**
3. ستجد تبويبات لإدارة كل قسم

## الأقسام القابلة للتعديل

### 1. رؤيتنا
- العنوان
- المحتوى
- اسم الأيقونة (Eye, Target, Award, etc.)

### 2. رسالتنا
- العنوان
- المحتوى
- اسم الأيقونة

### 3. أهدافنا الاستراتيجية
- يمكنك إضافة/تعديل/حذف الأهداف
- كل هدف يحتوي على:
  - النص
  - الترتيب

## الأيقونات المتاحة

الأيقونات المدعومة:
- `Eye` - العين (لرؤيتنا)
- `Target` - الهدف (لرسالتنا)
- `Award` - الجائزة
- `FileText` - الملف
- `Users` - المستخدمين
- `Heart` - القلب

## الملفات المحدثة

- `admin-cms/database/create_about_sections_table.sql` - SQL script
- `admin-cms/app/api/about/route.ts` - API لإدارة الأقسام
- `admin-cms/app/api/public/about/route.ts` - Public API
- `admin-cms/app/about/page.tsx` - صفحة الإدارة في CMS
- `admin-cms/lib/apiClient.ts` - إضافة وظائف API
- `src/lib/api.ts` - إضافة وظائف API للموقع العام
- `src/pages/About.tsx` - تحديث الصفحة لقراءة البيانات من API

## ملاحظات مهمة

1. **الحفظ**: بعد تعديل أي قسم، اضغط على زر **"حفظ جميع التغييرات"** في أسفل الصفحة
2. **الترتيب**: يمكنك تحديد `order_index` لترتيب الأهداف
3. **الأيقونات**: استخدم اسم الأيقونة باللغة الإنجليزية (مثل: Eye, Target)

## بعد التطبيق

1. شغّل SQL script لإنشاء الجداول
2. افتح CMS واذهب إلى "صفحة "من نحن""
3. عدّل المحتوى
4. احفظ التغييرات
5. افتح الموقع العام: `http://localhost:5173/about`
6. ستجد المحتوى محدثاً!
