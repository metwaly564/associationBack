# إعداد تصنيفات المشاريع

## نظرة عامة

تم إنشاء نظام لإدارة تصنيفات المشاريع من قاعدة البيانات. يمكنك الآن إضافة، تعديل، وحذف التصنيفات من CMS.

## الخطوات

### 1. إنشاء الجداول في قاعدة البيانات

قم بتشغيل ملف SQL التالي:

```powershell
psql -U postgres -d association_db -f admin-cms/database/create_project_categories_table.sql
```

هذا سينشئ:
- جدول `project_categories` - تصنيفات المشاريع
- عمود `category_id` في جدول `projects` (foreign key)

### 2. الوصول إلى صفحة الإدارة

1. سجّل الدخول إلى CMS: `http://localhost:3001`
2. من القائمة الجانبية، اختر **"المشاريع والبرامج"**
3. ستجد tab **"تصنيفات المشاريع"** لإدارة التصنيفات

## الميزات

### إدارة التصنيفات:
- **إضافة تصنيف جديد**: اضغط على "إضافة تصنيف جديد"
- **تعديل تصنيف**: اضغط على "تعديل" بجانب أي تصنيف
- **حذف تصنيف**: اضغط على "حذف" (لا يمكن حذف تصنيف مستخدم في مشاريع)

### حقول التصنيف:
- **الاسم**: اسم التصنيف (مثل: الرعاية الاجتماعية)
- **الرابط (Slug)**: رابط التصنيف (مثل: social-care)
- **الوصف**: وصف التصنيف (اختياري)
- **اللون**: لون التصنيف (للعرض في الواجهة)
- **اسم الأيقونة**: اسم الأيقونة (اختياري)
- **ترتيب العرض**: رقم الترتيب
- **الحالة**: نشط/غير نشط

### استخدام التصنيفات في المشاريع:
- عند إنشاء أو تعديل مشروع، ستجد قائمة منسدلة للتصنيفات بدلاً من حقل نص حر
- يمكنك اختيار تصنيف من القائمة أو تركها فارغة

## الملفات المحدثة

### قاعدة البيانات:
- `admin-cms/database/create_project_categories_table.sql` - SQL script

### API Routes:
- `admin-cms/app/api/project-categories/route.ts` - GET/POST لتصنيفات المشاريع
- `admin-cms/app/api/project-categories/[id]/route.ts` - GET/PUT/DELETE لتصنيف واحد
- `admin-cms/app/api/projects/route.ts` - تحديث لدعم `category_id`
- `admin-cms/app/api/projects/[id]/route.ts` - تحديث لدعم `category_id`
- `admin-cms/app/api/public/projects/route.ts` - تحديث JOIN مع التصنيفات

### CMS Pages:
- `admin-cms/app/projects/page.tsx` - إضافة tab لإدارة التصنيفات
- `admin-cms/app/projects/[id]/page.tsx` - تحديث لاستخدام Select بدلاً من Input
- `admin-cms/app/projects/new/page.tsx` - تحديث لاستخدام Select بدلاً من Input
- `admin-cms/lib/apiClient.ts` - إضافة دوال API لتصنيفات المشاريع

## بعد التطبيق

1. شغّل SQL script لإنشاء الجداول
2. افتح CMS واذهب إلى "المشاريع والبرامج" → tab "تصنيفات المشاريع"
3. أضف/عدّل التصنيفات حسب الحاجة
4. عند إنشاء أو تعديل مشروع، اختر التصنيف من القائمة المنسدلة
5. ستظهر التصنيفات في جدول المشاريع مع الألوان المخصصة

## ملاحظات مهمة

1. **الحذف**: لا يمكن حذف تصنيف مستخدم في مشاريع موجودة
2. **الترتيب**: يمكنك تحديد `order_index` لترتيب التصنيفات
3. **الألوان**: استخدم ألوان hex (مثل: #10b981) للتصنيفات
4. **الأيقونات**: يمكنك إضافة اسم أيقونة (مثل: Heart, Users) للتصنيف
