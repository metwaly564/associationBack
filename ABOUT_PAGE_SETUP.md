# إعداد صفحة "من نحن" لإدارتها من CMS

## نظرة عامة

تم ربط صفحة "من نحن" (`/about`) في الموقع العام بنظام إدارة المحتوى (CMS). الآن يمكنك تعديل محتوى الصفحة بالكامل من لوحة التحكم.

## الخطوات

### 1. إنشاء صفحة "من نحن" في قاعدة البيانات

قم بتشغيل ملف SQL التالي:

```powershell
psql -U postgres -d association_db -f admin-cms/database/create_about_page.sql
```

هذا سينشئ صفحة "من نحن" في جدول `static_pages` مع محتوى افتراضي.

### 2. الوصول إلى صفحة الإدارة

1. سجّل الدخول إلى CMS: `http://localhost:3001`
2. من القائمة الجانبية، اختر **"الصفحات"**
3. ابحث عن صفحة "من نحن" (slug: `about`)
4. اضغط على **"تعديل"**

### 3. تعديل المحتوى

في صفحة التحرير، ستجد التبويبات التالية:

#### تبويب المحتوى:
- **العنوان**: عنوان الصفحة الرئيسي
- **الرابط (Slug)**: يجب أن يكون `about`
- **نوع الصفحة**: اختر "من نحن"
- **الملخص**: ملخص قصير يظهر في Hero section
- **المحتوى (HTML)**: المحتوى الكامل للصفحة (يدعم HTML)

#### تبويب الوسائط:
- **صورة البانر**: يمكنك رفع صورة من الجهاز أو إدخال رابط

#### تبويب النشر:
- **الحالة**: اختر "منشورة" لنشر الصفحة

#### تبويب SEO:
- **عنوان الميتا**: للبحث
- **وصف الميتا**: للبحث

## استخدام HTML في المحتوى

يمكنك استخدام HTML لتنسيق المحتوى:

```html
<section class="container mx-auto px-4 py-16">
  <h2 class="text-3xl font-bold text-gray-900 mb-6">عنوان القسم</h2>
  <p class="text-gray-700 leading-relaxed">
    النص هنا...
  </p>
</section>
```

## الملفات المحدثة

- `admin-cms/app/api/public/pages/[slug]/route.ts` - Public API لجلب الصفحات
- `admin-cms/database/create_about_page.sql` - SQL script لإنشاء الصفحة
- `admin-cms/app/pages/[id]/page.tsx` - صفحة تحرير الصفحات (محدثة)
- `admin-cms/app/pages/new/page.tsx` - صفحة إنشاء الصفحات (محدثة)
- `src/lib/api.ts` - إضافة `getPageBySlug`
- `src/pages/About.tsx` - تحديث الصفحة لقراءة المحتوى من API

## ملاحظات مهمة

1. **الحالة**: تأكد من تعيين الحالة إلى "منشورة" حتى تظهر الصفحة في الموقع العام
2. **Slug**: يجب أن يكون `about` لتعمل الصفحة بشكل صحيح
3. **المحتوى الافتراضي**: إذا لم يتم تحميل المحتوى من API، سيتم عرض المحتوى الافتراضي المضمن في الكود
4. **HTML**: يمكنك استخدام HTML كامل في حقل المحتوى، بما في ذلك الأقسام، الفقرات، العناوين، والقوائم

## بعد التطبيق

1. شغّل SQL script لإنشاء الصفحة
2. افتح CMS واذهب إلى "الصفحات"
3. عدّل صفحة "من نحن"
4. احفظ التغييرات
5. افتح الموقع العام: `http://localhost:5173/about`
6. ستجد المحتوى محدثاً!
